--------Guide to get the website to run locally on your machine--------
-
-1) put the files in the following tree

├── app.py
├── plot_position_changes.py
├── static/
│   ├── class75.jpg
│   ├── script.js
│   └── style.css
└── webtemp/

-
-2) open powershell or equivalent in linux 
-3) cd to the F1 project folder 
-4) type : python app.py
-5) copy the dev server address and enjoy!
